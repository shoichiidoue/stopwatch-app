# ストップウォッチアプリ

シンプルで使いやすいWebベースのストップウォッチアプリケーション。

## 機能

- ⏱️ 時間、分、秒、ミリ秒の表示
- ▶️ スタート/ストップボタン
- 🔄 リセット機能
- 📊 ラップタイム記録

## 使い方

1. `index.html` をブラウザで開く
2. **スタート** ボタンをクリックしてタイマーを開始
3. **ストップ** ボタンをクリックしてストップ（ラップタイムが記録されます）
4. **リセット** ボタンでリセット

## ファイル構成

- `index.html` - HTMLマークアップ
- `style.css` - スタイルシート
- `script.js` - JavaScript機能
- `README.md` - このファイル
